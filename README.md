# Wikipedia RAG Agent

This project is a simple Retrieval-Augmented Generation (RAG) system.
It answers questions using Wikipedia text instead of model memory.

## Features
- Wikipedia ingestion
- Text chunking
- Vector search (FAISS)
- Groq LLM answers
- No hallucinations

## Change Topic
Edit the topic variable in run_demo.ipynb
